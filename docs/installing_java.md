---
title: Built-in Java Installer
description: Learn how to install Java with APKognito
---

APKognito provides tools for installing and configuring Java (JDK/JRE). This guide will show you how to install JDK 25 using the APKognito Java Installer.

JDK can always be installed from the [Oracle Java Downloads](https://www.oracle.com/java/technologies/downloads/) page.
